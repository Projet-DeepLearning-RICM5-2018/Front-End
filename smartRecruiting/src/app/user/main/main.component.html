SmartRecruiting

Un outil pour accélérer votre recrutement d'étudiants Grenoblois

Ce site se base sur des algorithmes de deep learning pour prédire quelle formation grenobloise répondra au mieux à votre offre de stage, offre d'emploi ou description de projet.

  • Fonctionnement du site

    1. Entrez une offre sous format texte pour en obtenir la prédiction.
    2. Inscrivez-vous ou connectez vous pour pouvoir retrouver cette offre dans l'onglet Mon compte > Mes offres
    3. Lorsque vous aurez trouvé le profil adapté à votre offre vous pourrez donner un feedback dans l'onglet contenant vos offres pour nous aider à améliorer notre base de données et aider SmartRecruiting à vous fournir toujours de meilleures prédictions.
